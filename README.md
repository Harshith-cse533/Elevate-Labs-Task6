# Task 6 - K-Nearest Neighbors (KNN) Classification

## Objective
To understand and implement the K-Nearest Neighbors (KNN) algorithm for classification problems using the Iris Dataset.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Steps Performed

1. Loaded the Iris Dataset.
2. Split the dataset into training and testing sets.
3. Normalized the features using StandardScaler.
4. Trained a KNN classifier.
5. Predicted test data labels.
6. Evaluated model performance using accuracy score.
7. Generated a confusion matrix.
8. Experimented with different K values.
9. Plotted Accuracy vs K graph.
10. Visualized decision boundaries.

## Results

- Successfully implemented KNN Classification.
- Achieved high classification accuracy on the Iris Dataset.
- Observed the impact of different K values on model performance.
- Visualized classification regions using decision boundaries.

## Key Concepts Learned

- Instance-Based Learning
- Euclidean Distance
- Feature Normalization
- K Value Selection
- Confusion Matrix
- Classification Accuracy
- Decision Boundaries

## Conclusion

K-Nearest Neighbors (KNN) is a simple yet effective classification algorithm that classifies data points based on the majority class among their nearest neighbors. Proper normalization and choosing an appropriate K value significantly improve model performance.
