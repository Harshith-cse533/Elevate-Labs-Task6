K-Nearest Neighbors (KNN) Classification

Objective

To understand and implement the K-Nearest Neighbors (KNN) algorithm for classification using the Iris Dataset.

Tools Used

- Python
- Pandas
- Scikit-learn
- Matplotlib

Task Steps

1. Loaded the Iris Dataset.
2. Normalized features using StandardScaler.
3. Split the dataset into training and testing sets.
4. Trained a KNN classifier.
5. Tested multiple K values.
6. Evaluated model performance using Accuracy Score.
7. Generated a Confusion Matrix.
8. Visualized the Decision Boundary.

K Values Tested

- K = 1
- K = 3
- K = 5
- K = 7
- K = 9

Results

The model was evaluated using accuracy score and confusion matrix. Among the tested values, K = 5 provided the best classification accuracy on the Iris Dataset.

Files Included

- "knn_classification.py"
- "confusion_matrix.png"
- "decision_boundary.png"
- "requirements.txt"

Learning Outcomes

- Understanding the KNN algorithm.
- Importance of feature scaling in distance-based algorithms.
- Selecting an appropriate K value.
- Evaluating classification models using accuracy score and confusion matrix.
- Visualizing decision boundaries for classification tasks.
